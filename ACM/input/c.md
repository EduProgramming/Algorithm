# C 입출력

## 데이터 예제 입력이 하나일 때

> **예제 문제**
>
> [https://www.acmicpc.net/problem/5585](https://www.acmicpc.net/problem/5585)
>
> **답안 소스**
>
> [정답 소스코드](../greedy/p5585.%20%EA%B1%B0%EC%8A%A4%EB%A6%84%EB%8F%88.c)

### 입출력

```c
#include<stdio.h>
#include<stdlib.h>

int main() {
    int N;
    scanf("%d", &N);

    printf("%d", N); // 출력
    return 0;
}
```

## 데이터 예제 입력 여러개일 때

> **예제 문제**
>
> [https://www.acmicpc.net/problem/2798](https://www.acmicpc.net/problem/2798)
>
> **답안 소스**
>
> [정답 소스코드](../brute_force/p2798.%20%EB%B8%94%EB%9E%99%EC%9E%AD.c)

### 입출력

```c
#include<stdio.h>
#include<stdlib.h>

int main() {
    int N, M;
    scanf("%d %d", &N, &M);

    return 0;
}
```
